Los tres pilares de la Web

Tim Beerne-Leees considerado el padre de la Web porque desarrolló los tres elementos básicos para el funcionamiento de la Web

HTML

HTTP

URL

HTML

HTMl, siglas de HyperText Markup Language, hace referencia marcado para is elaboración de páginas web. Es un estándar que sirve de referencia para la elaboración de páginas web en sus diferentes verdeluctua básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, en estudar a cargo de la W3C. organización dedicada a la estandarización de casi todas las tecnologias ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Es el lenguaje con el que se definen las páginas web


Fuente HTML, Wikipedia

HTTP

Hypertext Transfer Protocol o HTTP (en español protocolo de transferencia de hipertexto) es el protocolo usado en cada transacción de la World Wide Web. HTTP fue desarrollado por el World Wide Web Consortium y la Internet Engineering Task Force, colaboración que culminó en 1999 con la publicación de una serie de RFC, el más importante de ellos es el RFC 2616 que especifica la versión 1.1.

Fuente: HTTP

URL

Un localizador de recursos uniforme o URL siglas en ingles de Uniform Resource Locator es un identificador de recursos uniforme (URI) cuyos recursos referidos pueden cambiar, esto es, la dirección puede apuntar a recursos variables en el tiempo. Están formados por una secuencia de caracteres, de acuerdo a un formato modelico y estándar, que designa recursos en una red, como Internet.
